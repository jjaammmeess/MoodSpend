情绪消费记账APP完整原型思路方案
一、APP 核心定位
产品名称：
中文：花钱了
英文：AfterBuy
- SwiftUI
- MVVM
- SwiftData
- CloudKit
- 支持Dark Mode
- 仅支持iPhone竖屏
- 中英文本地化
- 本地规则引擎
 核心 slogan：记的不是花销，是花钱时的心情 产品定位：一款聚焦「消费金额 + 消费情绪 / 动机」双维度记账工具，拒绝单纯省钱说教，通过记录每笔消费背后的情绪，帮用户洞察消费心理、告别冲动 / 面子消费，找到真正让自己快乐的花钱方式，主打极简记账、情绪复盘、心理洞察，面向 18-35 岁年轻群体（学生、职场人）。 核心差异：市面上记账 APP 只做收支统计，本品核心做情绪 - 消费关联分析，主打自我洞察，而非理财省钱。
二、产品核心架构
采用底部 4 栏 tab 导航，操作极简，符合用户使用习惯，分别为：首页（快速记账 + 今日概览）、账单列表、情绪复盘、我的。
三、各页面完整原型设计
（一）Tab1：首页（核心记账页，3 秒极速记账）
	•	页面布局
	•	顶部：今日日期 + 今日总支出 + 今日情绪消费概括（例：今日 3 笔开销，1 笔取悦自己，2 笔刚需）
	•	中部：极速记账按钮（醒目的圆形 + 按钮，标注「记一笔」，点击弹出记账弹窗）
	•	下部：今日消费明细卡片（单条展示：金额 + 消费类目 + 情绪标签，极简排版）
	•	右上角：快速筛选 / 搜索按钮
	•	核心记账弹窗（最重要页面）
	•	第一步：输入金额（数字键盘，默认支出，可切换收入）
	•	第二步：选择消费类目（刚需日用、餐饮、服饰、娱乐、社交、交通等，支持自定义）
	•	第三步：选择情绪 / 动机标签（核心功能） 固定预设标签： ✅ 取悦自己（真心喜欢、提升自我） ✅ 刚需必要（生活必需、不得不花） ✅ 冲动消费（一时上头、无计划） ✅ 解压发泄（压力大、心情不好花钱） ✅ 面子社交（请客、攀比、碍于情面） ✅ 仪式感（节日、犒劳自己、纪念） 支持 + 自定义标签，添加专属情绪
	•	第四步：可选操作（写 10 字内备注、上传消费小票 / 物品照片）
	•	底部：保存按钮，一键完成记账
（二）Tab2：账单列表
	•	页面布局
	•	顶部：时间筛选（日 / 周 / 月 / 年）+ 支出 / 收入筛选
	•	主体：按时间倒序排列消费账单，每条卡片展示： 日期 + 时间 + 金额 + 消费类目 + 情绪标签（高亮突出） + 备注 / 照片缩略图
	•	交互：点击单条账单，可编辑 / 删除；长按可快速操作
（三）Tab3：情绪复盘（核心特色页）
	•	月度概览
	•	顶部：月度总支出 + 情绪消费占比饼图（直观看到面子、冲动、解压等无效消费占比）
	•	情绪消费排行榜：本月花钱最多的情绪动机（例：解压消费占比 35%，排名第一）
	•	深度分析模块
	•	消费规律：例「每周三晚易产生解压消费」「每月社交开销集中在周末」
	•	快乐消费对比：取悦自己 / 仪式感消费 vs 冲动 / 面子 / 解压消费金额对比
	•	消费提醒：针对高频无效消费给出温柔提示（例：本月解压消费 12 笔，试试换种方式放松～）
	•	日历视图
	•	每日情绪消费标记，一眼看清整月花钱心情
（四）Tab4：我的
	•	个人信息：头像、昵称、记账天数统计
	•	数据管理：账单导出、数据备份 / 恢复、清除数据 3 - 自定义设置：新增消费类目、新增情绪标签、记账提醒开关
	•	其他：使用帮助、意见反馈、关于我们
四、核心特色功能原型细节
	•	情绪消费预警 当用户短期内多次记录「冲动 / 解压 / 面子」消费，APP 弹出温柔提示，而非强制劝阻，文案温和：「最近是不是压力有点大呀，慢下来想想这笔钱是不是真的想花～」
	•	月度情绪消费报告 每月 1 号生成专属报告，包含：情绪消费占比、消费心理洞察、省钱小建议、快乐消费建议，支持图片分享
	•	无压力记账 极简操作，无繁琐步骤，不设置预算焦虑、不强制省钱，纯记录洞察，降低用户使用门槛
	•	数据隐私 支持本地存储 + 云端加密备份，保障用户消费及情绪数据安全
五、交互逻辑原型
	•	打开 APP 直接进入首页，无需复杂注册登录（支持游客体验，登录仅用于数据备份）
	•	记账全程最多 4 步，无多余页面跳转，3 秒完成
	•	情绪标签采用彩色小标签设计，不同情绪对应不同浅色系（取悦自己 - 粉色、刚需 - 蓝色、冲动 - 灰色、解压 - 黄色、面子 - 紫色），视觉直观区分
	•	复盘页面图表可视化，文字简洁，无专业理财术语，通俗易懂
六、开发落地优先级
第一阶段（基础核心版，快速上线）
	•	首页极速记账功能
	•	基础账单列表
	•	预设情绪标签 + 情绪消费统计
	•	我的（基础数据管理）
第二阶段（迭代优化版）
	•	月度深度复盘分析
	•	自定义情绪 / 类目标签
	•	记账提醒、消费预警
	•	数据云端备份
第三阶段（增值特色版）
	•	年度情绪消费报告
	•	消费心理小科普
	•	账单导出、分享功能
七、UI 设计风格建议
整体采用极简温柔风，浅色系为主，无刺眼色彩，界面干净无广告，字体清晰，卡片式布局，突出情绪标签视觉效果，营造「树洞式」温暖感，贴合用户记录情绪的心理需求。
